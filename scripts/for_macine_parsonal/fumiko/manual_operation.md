# 手動セットアップ手順

どうしても「自動にできなかった部分」について、手動でやった操作につきここに記述していく。

## インストール時

最新のDebianをUSBから入れる。

インストーラに従い、ほぼデフォルトで入れたが、選んだとこといえば

+ 日本語
+ Host名は"fumiko"に
+ ローカルドメインは"local.sumpic.orz.hm"
+ リポジトリは国内
+ パーティションはSSD一個だけつないでパーティション２つ(ふたつ目はSwap)にウィザードで決定
+ デフォルトインストールパッケージはすべてをクリアした上で
  + SSHサーバ
  + 標準システムユーティリティ
