# raspaberypi_for_xfd

## What's this ?

RaspberyPiに(USB機器を電源管理出来る)USBハブと音声とJenkinsという「XFDの元」環境を作成する。

## Target

+ 対象OS(LinuxDistoribution) : Raspabian
+ 対象機器 : RaspaberyPi2
